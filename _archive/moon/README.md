A CLI from the moon, beamed directly to your terminal

```
Usage:
  moon [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  new         Create a new moon folder inside working dir
  orbit       Choose a command from moon.config.json to run against
  readMe      Read all files in the directory

Flags:
  -h, --help                help for moon
      --openAPIKey string   OPENAI API KEY [Required]

Use "moon [command] --help" for more information about a command.
```
